# median
Program to calculate the Median of list of data
Приложение Median.
Если его просто запустить из командной строки, то оно сгенерирует 1000 пробных данных
в файл data.csv .
Далее, приложение проведет анализ этих данных на медианную, и загрузит данные в папку
results.csv .

Если в командной строке дать аргумент config.toml (как пример названия файла конфигурации)
, то приложение будет искать файлы
data.csv и создаст файл results.csv в указанных в файле config.toml директориях.
Далее, оно проведет анализ данных из файла data.csv и загрузит данный в файл
results.csv (с медианной).

Пример файла config.toml:
[main]
    input './data'
    output './data/results'

здесь приложение ищет файл data.csv  в паке ./data и загружает данные с медианной
в папку ./data/results в файл results.csv

в приложении есть error-checking, поэтому если не все данные четкие, приложение все равно
проведет анализ медианной.

приложение само создаст папку ./data/results, но папка ./data должна быть Вами создана,
вместе с файлом data.csv с данными. Файл надо поместить в папку ./data перед запуском
программы.
